# Data-Driven Analysis of Aadhaar Enrolment Patterns in India

## Overview
This project analyzes Aadhaar enrolment data published by UIDAI to identify patterns across states, districts, age groups, and time periods.

## Objectives
- Analyze state-wise and district-wise enrolment trends
- Study age-group-wise enrolment distribution
- Identify temporal variations in enrolment activity
- Provide data-driven insights for policy planning

## Dataset
Source: UIDAI Open Government Data Platform  
Type: Aadhaar Monthly Enrolment Data

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- Significant inter-state and intra-state enrolment variations
- Higher enrolments observed in adult (18+) category
- Temporal fluctuations indicate varying enrolment activity
